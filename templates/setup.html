How to Create Cloudflare API Token

Step-by-step instructions:

  1. 1.

    Login to Cloudflare

    Go to your Cloudflare dashboard

  2. 2.

    Navigate to API Tokens page

    Go to https://dash.cloudflare.com/profile/api-tokens

  3. 3.

    Create Custom Token

    Click "Create Token", then "Create Custom Token"

  4. 4.

    Set Permissions

    Add the following permissions:

    • ✓ Account, Account Settings, Read
    • ✓ Account, Cloudflare Tunnel, Edit
    • ✓ Zone, Zone, Read
    • ✓ Zone, DNS, Edit
  5. 5.

    Complete Token Creation

    Click "Continue to summary" and then "Create Token"

warning

Important:

Copy and save your API token immediately after creation. You won't be able to see it again.
